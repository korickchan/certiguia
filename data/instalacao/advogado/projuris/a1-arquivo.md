## Configurar e-CPF A1 em arquivo no Projuris

Certificado **A1 .pfx** instalado no **computador** onde você usa o Projuris.

### Pré-requisitos

- Arquivo `.pfx` / `.p12` e senha recebidos após emissão do e-CPF A1.

### Importar no Windows

1. Dê duplo clique no `.pfx` ou use `certmgr.msc` → **Pessoal** → **Importar**.
2. Informe a senha da certificadora.

### No Projuris

1. **Configurações** → **Certificado digital**.
2. Selecione **Certificado A1** ou **Arquivo** / **Windows store**.
3. Escolha o certificado e-CPF listado ou aponte para o `.pfx`.
4. Informe a senha quando o Projuris solicitar (pode marcar «lembrar neste computador»).

### Testar

Assine documento de teste e valide o PDF no ITI.

### Limitação

A1 em arquivo **não é indicado para celular**. Para assinar no smartphone, use A3 nuvem ou troque o perfil no CertiGuia.
